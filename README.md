# Static Website Hosting on AWS S3

## Project Overview
This project demonstrates how to host a simple static website using AWS services. The website consists of HTML and CSS files and is deployed on AWS.

## Technologies Used
- HTML5
- CSS3
- AWS

## Project Files
- index.html – Main webpage
- style.css – Stylesheet

## Features
- Responsive static website
- Clean and simple design
- Hosted on AWS

## Learning Outcomes
- Created a static website using HTML and CSS.
- Learned the basics of AWS static website hosting.
- Uploaded project to GitHub for version control.
